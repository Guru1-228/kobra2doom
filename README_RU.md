# KobraDoom — Anycubic Kobra 2 Pro

https://youtu.be/vQZF1pdODIo?si=UrtsiVXG3oMMBpgu

Форк проекта [milkpirate/kobradoom](https://github.com/milkpirate/kobradoom), адаптированный специально для 3D-принтера **Anycubic Kobra 2 Pro**.

### Важное отступление от автора
> Я не программист, а просто энтузиаст, который захотел запустить DOOM на своём 3D-принтере. Весь код адаптировался при плотной поддержке со стороны ИИ. У меня не было цели сделать «всё правильно и по канонам» — главной задачей было заставить игру запускаться и работать. 

## На чём это тестировалось
* **Принтер:** Anycubic Kobra 2 Pro
* **Прошивка:** Стоковая версия **2.3.9** (с полученным root-доступом).
* *Примечание:* Тестировалось **только** на этой конфигурации. Есть вероятность, что заведётся и на других прошивках/версиях принтеров Anycubic, но гарантий нет.

## Зачем нужен был форк?
Оригинальный проект `milkpirate/kobradoom` при запуске на моём принтере выдавал просто чёрный экран. Чтобы исправить это и адаптировать игру под Kobra 2 Pro, я внёс следующие изменения:
1. **Вывод изображения:** Изменён код вывода графики, чтобы картинка корректно отображалась на экране принтера.
2. **Разрешение:** Адаптировано под родное разрешение дисплея Cobra 2 Pro.
3. **Управление:** Ввод привязан к `/dev/input/event1` (подключенная USB-клавиатура).

## Как запустить
Вам понадобится сам исполняемый файл и файл ресурсов игры (WAD).

1. Подключите обычную USB-клавиатуру напрямую в USB-порт принтера.
2. Поместите исполняемый файл `kobradoom` и файл `doom1.wad` в одну папку на принтере (например, через SSH или SFTP).
3. Подключитесь к принтеру по SSH.
4. Перейдите в папку с файлами и запустите игру:
   ```bash
   ./kobradoom
   ```

## Сборка из исходников
Если вы хотите скомпилировать проект самостоятельно, вам потребуется настроенный кросс-компилятор под ARM. Я собирал это на своём ПК (Linux) с использованием тулчейна для процессоров Rockchip.

**Команды для сборки:**
```bash
cd doomgeneric
make -f Makefile.kobra LDFLAGS="-static" CC=arm-rockchip830-linux-uclibcgnueabihf-gcc CXX=arm-rockchip830-linux-uclibcgnueabihf-g++
```
